Diracian Physics Calculator

Version 1.3.2

Tired of repetitive physics homework with the same pointless work over and over again? Have a lot of numbers to crunch in a short period of time? The Diracian Physics Calculator is the solution to your problems! It is a free, easy to use, program to automatically crunch all of the numbers in your physics problems with no effort needed from you!


- It is recommended that you have a basic knowledge of physics before using this program.
- While inputing data, be sure to use standard SI units. Not doing so will cause problems in equations that make use of constants.
- If the program is failing to run, make sure you are using the most up to date version of Chrome, Opera, or Safari.
